# hotel-magement-dashboard-excel

### Project Overview

This project involves the design and development of an interactive **Hotel Management Dashboard** using Microsoft Excel for data analysis and reporting. The dashboard provides hotel management with a clear, near real-time overview of operational performance, profitability, and trends to support data-driven decision-making.

The focus of the project was not only on visualization, but also on **automation, usability, and business relevance**.

---

### Tools & Technologies

* Microsoft Excel (Advanced)
* Pivot Tables & Pivot Charts
* Excel Formulas
* VBA (Macros)
* Slicers & Interactive Controls

---

### Key Performance Indicators (KPIs)

* **Hotel Count** – Total number of hotels in the dataset
* **Total Profit** – Overall profit generated across all hotels
* **Rooms Available** – Total available rooms
* **Total Bookings** – Number of bookings recorded
* **Total Customers** – Number of customers served
* **City Count** – Number of unique cities represented
* **Country Count** – Number of unique countries represented

These KPIs provide a high-level operational, financial, and geographic overview of hotel performance and capacity utilization.

* **Rooms Booked by Country (Donut Chart)** – Shows the distribution of bookings across countries
* **Room Availability by City (Radar Chart)** – Compares room availability across different cities
* **Customer Type Distribution (Donut Chart)** – New vs Existing customers
* **Monthly Profit Trends (Line Chart)** – Identifies seasonality and performance trends over time
* **Rooms Available vs Rooms Booked (Horizontal Bar Chart)** – Compares supply versus demand
* **Top Booked Hotels (Funnel Chart)** – Highlights hotels with the highest booking volumes

Each visualization was selected to answer specific business questions related to operational efficiency, demand analysis, customer behavior, and financial performance.

---

### Interactivity & Automation

* **Year Slicer** for time-based analysis and trend comparison
* **VBA Macro Button** that refreshes the entire dashboard automatically whenever the data is updated, enabling real-time reporting and reducing manual effort

---

### Business Value

* Enables data-driven decision-making through clear KPI reporting
* Improves operational visibility across hotels, cities, and countries
* Supports profitability analysis and trend identification
* Reduces manual reporting through dashboard automation
